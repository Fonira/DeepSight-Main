## Contexte

DeepSight integre les scores Tournesol depuis plusieurs mois sur web, mobile et extension :
- Proxy CORS backend vers api.tournesol.app (5 endpoints)
- Widget avec 10 criteres sur chaque analyse
- Section "Recommandations Tournesol" sur le dashboard
- 20% du poids dans l'algorithme de decouverte
- Badge sunflower avec deep link vers tournesol.app
- Attribution dans CREDITS.md, footer, pages About

La base de donnees Tournesol est sous licence ODC-By (reutilisation commerciale autorisee avec attribution). L'API est publique, sans authentification. Pas de rate limits documentes.

Strategie detaillee : `docs/STRATEGIE-TOURNESOL-PARTENARIAT.md`
