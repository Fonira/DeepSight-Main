Semaine 3-4

3. Agents API — Remplacer Perplexity + Brave

Créer un Agent Mistral "DeepSight Analyst" avec web search natif. Un seul appel = analyse enrichie + fact-check + sources web. Supprime la dépendance Perplexity et simplifie le code.

Mois 2

4. Voxtral TTS — Audio Summaries

Intégrer Voxtral TTS pour les synthèses audio. Voix DeepSight brandée (voix française). Le AudioSummaryPlayer existe déjà côté frontend — il manque juste le backend TTS.

Mois 2

5. Voxtral STT — Unifier les fallbacks transcription

Remplacer Groq Whisper, OpenAI Whisper, Deepgram, AssemblyAI par un seul provider : Voxtral STT. Simplifie la Phase 3 du pipeline de transcription.

Mois 3

6. Fine-tuning — Modèle DeepSight custom

Exporter les ~500+ analyses existantes en JSONL. Fine-tuner mistral-small sur le format/ton/structure DeepSight. Résultat : qualité Large pour le prix de Small. 10x réduction coûts analyses.

Mois 3-4

7. Document AI — Analyse de PDFs

Feature "Upload un PDF/doc → DeepSight l'analyse". Cible : étudiants, chercheurs. Utilise Mistral OCR pour le parsing, puis le pipeline d'analyse classique.

Mois 4+

8. Mistral Small 4 — A/B Test + Migration

Benchmarker Small 4 vs Medium/Large sur un sample d'analyses. Si performances équivalentes → migrer les plans Pro vers Small 4 pour 3x savings en output.
